## Tipos de Dato
Cada variable puede tener un tipo de dato asociado.
- Integer (int)
- Float (float)
- String (str)
- Booleans (bool)

## Ejercicio

- Ejemplo: `harina = 3 #Variable: harina, Tipo de dato: int`
`huevos = 6 #Variable: huevos, Tipo de dato: int`
`leche = 1.5 #Variable: leche, Tipo de dato: float`
